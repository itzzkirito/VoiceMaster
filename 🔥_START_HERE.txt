========================================

If the server doesn't start:

1. Check Java:
   - Open Command Prompt
   - Type: java -version
   - Should show version 8 or higher

2. Check port:
   - Another program might be using port 2030
   - Close other applications
   - Or change port in application.yml (line 190)

3. Check logs:
   - After starting, check: lavalink\logs\ folder
   - Look for error messages

4. Run diagnostic:
   - Double-click: lavalink\check-setup.bat
   - This will tell you what's wrong
